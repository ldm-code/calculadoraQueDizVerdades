 # Calculadora Que Diz Verdades

 - Um projeto desenvolvido em JavaScript que consiste em uma calculadora basica que mostra uma mensagem diefernte quando o resultado e igual a 67,com o intuito de estudar os conceitos da
linguagem e como ela afeta paginas html

---

## Funcionalidades

- Realiza operações matemáticas básicas: adição, subtração, multiplicação e divisão.

- Exibe uma mensagem quando o numero 67 e ativado.

- Estilizacao basica com css

- Organizacao basica de pastas,separando o html do css.

---

## Tecnologias

- Javascript
  
- html
  
- css
